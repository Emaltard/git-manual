Git remote remove

Git remote remove est utilisé afin de supprimer ce que le remote lui même a fait. Vous avez fait un remote et donc donné un nom à ce remote. Si vous voulez supprimer ce que vous venez de faire avec le remote, il faut juste taper git remote remove dans la console afin de supprimer cela.

Syntaxe

git remote remove (et le nom du remote)

Affichage

$ git remote remove def** origin upstream $ git remote remove upstream $ git remote defeo origin
