Git rebase

Grâce a la commande git rebase, en considérant cinq branches par exemple toutes les modifications introduites et validées sur une des branches seront totalement appliquées sur les autres branches. Donc on peut dire que les branches en question sont au même niveauet que les branches secondaires sont au même niveau que la branche principale.

Syntaxe

$ git rebase
