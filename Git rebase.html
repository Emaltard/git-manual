Git rebase

Grâce a la commande git rebase, en considérant cinq branches par exemple toutes les modifications introduites et validées sur une des branches seront totalement appliquées sur les autres branches. Donc nous pourront dire que les branches en question sont au même niveau.

Syntaxe

$ git log --stat
