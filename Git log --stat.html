Git log --stat

La commande git log --stat affiche une liste des denières modifications (commits), ce qui permet de voir les les lignes supprimées ou ajoutées ou bien de voir et cette commande permet aussi de voir combien de lignes ont été changées et combien de lignes ont été ajoutées

Syntaxe

$ git log --stat
