	<h1 id="titreCommande">Git log --stat</h1>
	<p>La commande git log --stat affiche une liste des dernières modifications (commits) avec l'auteur de la modification.
       Cette commande nous permet de voir les lignes supprimées , ajoutées ou bien modifiées. Cette commande indique aussi 
	   la date et l'heure de chaque modification.</p>
	<h2>Syntaxe</h2>
	<p>$ git log --stat</p>
