git fetch -v

Cette commande importe des commits à partir d'un répo à distance dans votre répo local. Les commits résultants sont stockés sous forme de branches à distance à la place des branches locales avec lesquelles vous avez travaillé.

git fetch -t

Cette commande permet de "fetch" tous les tag des repertoires distant. Attention: git fetch -t ne fait pas la même fonction que la commande git fetch.

Dans l'exemple ci-dessous, on sera dans le cas où j'ai modifié des fichiers dans le repertoires distants et je n'ai pas pull les changements(dernier commit dans le repertoire distant: +2).

Syntaxe

git fetch -t
