git push -f

Habituellement, la commande refuse de mettre à jour un arbitre à distance qui n'est pas un ancêtre de l'arbitre local utilisé pour l'écraser. Aussi, lorsque l'option --force-avec-bail est utilisé, la commande refuse de mettre à jour un arbitre à distance dont la valeur actuelle ne correspond pas à ce qui est attendu.

Syntaxe

git push --f
